Introduction

Lister is a responsive template powered by Twitter Bootstrap Framework. Booster has a simple yet powerful set of UI elements and plugins to get your work done in the fastest time possible, with a user friendly and clean design to appeal to your customers.

We really care for our buyers and so in case if you have any question or feedback, please feel free to get back to us.

Structure

We have tried to follow the standards and modular structure while developing the theme. Following sections are explaining the theme File & Folder, structure, html file struture and js.

File & Folder Structure


                            ├── Booster main folder
                            ├── dist/css/css/
                            │   └── All css files.
                            ├── dist/images/images/
                            │   └── All images
                            └── assets/js/
                                └── All Javascripts files
                            └── dist/js/js/
                                └── All Plugin JS files

HTML Structure

CSS

This template using the lot of css files, some of are from third parties. The main css files are plugins.min.css, main.min.css, these files are using into all html files and must be included to work it properly.

Files are explained below:

File Description
plugins.min.css The core plugins file is being used in all the pages.
main.min.css We are using the single css file for this template, this file is main css file of this template.

Javascript

Booster uses jQuery, Bootstrap 4 JS framework(at its core) and some of the third-party plugins. There are may more third party plugin which you can use according to your needs. The css is already containing matching style for these plugins so you will not need to do anything around it.

Files are explained below:

File Description
plugins.min.js These files are used at core of the theme.
common.js You can use this file for add any jquery scripts
